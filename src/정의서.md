### 📘 블로그 테이블 (BlogEntity)
| 필드명         | 타입                  | 설명                                       |
| ----------- | ------------------- | ---------------------------------------- |
| `id`        | Long                | 게시글 고유 ID (PK, 자동 생성)                    |
| `title`     | String              | 게시글 제목                                   |
| `content`   | String              | 게시글 내용 (TEXT)                            |
| `createdAt` | LocalDateTime       | 작성일시                                     |
| `comments`  | List<CommentEntity> | 게시글에 달린 댓글 목록 (`@OneToMany`, cascade 적용) |

### 📘 댓글 테이블 (CommentEntity)
| 필드명         | 타입            | 설명                                           |
| ----------- | ------------- | -------------------------------------------- |
| `id`        | Long          | 댓글 고유 ID (PK, 자동 생성)                         |
| `writer`    | String        | 댓글 작성자                                       |
| `content`   | String        | 댓글 내용                                        |
| `createdAt` | LocalDateTime | 댓글 작성일시                                      |
| `blog`      | BlogEntity    | 댓글이 속한 블로그 게시글 (`@ManyToOne`, FK: `blog_id`) |

### 📘 시작 API 정의서 (IndexController)
| 기능         | Method | 경로  | 설명              | 요청 파라미터 / 요청 바디 | 반환 / 처리                     |
| ---------- | ------ | --- | --------------- | --------------- | --------------------------- |
| **시작 페이지** | GET    | `/` | 애플리케이션 루트 요청 처리 | 없음              | 블로그 목록 페이지(`/blogs`)로 리다이렉트 |


### 📘 블로그 API 정의서 (BlogController)
| 기능         | Method | 경로                   | 설명               | 요청 파라미터 / 요청 바디                        | 반환 / 처리                                     |
| ---------- | ------ | -------------------- | ---------------- | -------------------------------------- | ------------------------------------------- |
| **전체 조회**  | GET    | `/blogs`             | 모든 블로그 게시글 목록 조회 | 없음                                     | `blog/list.html` 뷰 렌더링, 모델에 `blogs` 전달      |
| **단건 조회**  | GET    | `/blogs/{id}`        | 특정 블로그 게시글 조회    | `id` (PathVariable)                    | `blog/detail.html` 뷰 렌더링, 모델에 `blog` 전달     |
| **글 작성 폼** | GET    | `/blogs/new`         | 블로그 글 작성 폼 페이지   | 없음                                     | `blog/write.html` 뷰 렌더링, 모델에 빈 `BlogDTO` 전달 |
| **글 등록**   | POST   | `/blogs`             | 블로그 글 등록         | `BlogDTO` (폼 데이터)                      | 등록 후 `/blogs`로 리다이렉트                        |
| **글 수정 폼** | GET    | `/blogs/{id}/edit`   | 블로그 글 수정 폼 페이지   | `id` (PathVariable)                    | `blog/edit.html` 뷰 렌더링, 모델에 `blog` 전달       |
| **글 수정**   | POST   | `/blogs/{id}`        | 블로그 글 수정         | `id` (PathVariable), `BlogDTO` (폼 데이터) | 수정 후 `/blogs/{id}`로 리다이렉트                   |
| **글 삭제**   | POST   | `/blogs/{id}/delete` | 블로그 글 삭제         | `id` (PathVariable)                    | 삭제 후 `/blogs`로 리다이렉트                        |

### 📘 댓글 API 정의서 (CommentController)
| 기능             | Method | 경로                          | 설명                   | 요청 파라미터 / 요청 바디            | 반환 / 처리                                |
| -------------- | ------ | --------------------------- | -------------------- | -------------------------- | -------------------------------------- |
| **블로그별 댓글 조회** | GET    | `/api/comments/{blogId}`    | 특정 블로그 게시글의 댓글 목록 조회 | `blogId` (PathVariable)    | JSON 형태의 댓글 목록 반환 (`List<CommentDTO>`) |
| **댓글 등록**      | POST   | `/api/comments`             | 새로운 댓글 등록            | `CommentDTO` (RequestBody) | 저장된 댓글 정보(JSON, `CommentDTO`) 반환       |
| **댓글 삭제**      | DELETE | `/api/comments/{commentId}` | 특정 댓글 삭제             | `commentId` (PathVariable) | 삭제 처리 (응답 없음)                          |


### 📘 블로그 서비스 로직 정의서 (BlogService)
| 함수 이름      | 매개변수                       | 반환형             | 설명                     | 특징 / 비고                                              |
| ---------- | -------------------------- | --------------- | ---------------------- | ---------------------------------------------------- |
| `findAll`  | 없음                         | `List<BlogDTO>` | 전체 게시글 조회 후 DTO 리스트 반환 | DB 조회 후 Entity → DTO 변환                              |
| `findById` | `Long id`                  | `BlogDTO`       | 단일 게시글 조회 후 DTO 반환     | 게시글이 존재하지 않으면 `IllegalArgumentException` 발생          |
| `save`     | `BlogDTO blogDTO`          | `Long`          | 새 게시글 등록 후 저장된 ID 반환   | `createdAt` 현재 시각으로 설정, Entity → DTO 매핑              |
| `update`   | `Long id, BlogDTO blogDTO` | `void`          | 게시글 제목/내용 수정           | 게시글 존재 여부 확인, `createdAt`은 유지                        |
| `delete`   | `Long id`                  | `void`          | 게시글 삭제                 | 게시글 존재 여부 확인 후 삭제, 없으면 `IllegalArgumentException` 발생 |

### 📘 댓글 서비스 로직 정의서 (CommentService)
| 함수 이름          | 매개변수                    | 반환형                | 설명                            | 특징 / 비고                                                                |
| -------------- | ----------------------- | ------------------ | ----------------------------- | ---------------------------------------------------------------------- |
| `findByBlogId` | `Long blogId`           | `List<CommentDTO>` | 특정 블로그의 댓글 목록 조회 후 DTO 리스트 반환 | Entity → DTO 변환, 블로그 ID 수동 매핑                                          |
| `save`         | `CommentDTO commentDTO` | `CommentDTO`       | 댓글 등록 후 저장된 DTO 반환            | 블로그 존재 여부 확인, DTO → Entity 변환, 작성일시(LocalDateTime.now()) 설정, 블로그 참조 설정 |
| `delete`       | `Long commentId`        | `void`             | 댓글 삭제                         | 댓글 존재 여부 확인, ID 기반 삭제, 없으면 `IllegalArgumentException` 발생               |
